Using the Application

This GIS web application allows stakeholders to submit their transportation related ideas for improving the North Lake Shore Drive multi-modal corridor. It is a single application that can be used on a desktop browser, mobile phone, and tablet device. Pan and zoom around, click/tap directly on the map to pin your comment to the map. Then, select a purpose & need category that your idea addresses, provide details about your comment, input your zip code and click/tap 'Submit' to finalize your request. Providing your zip code will allow the project team to understand the concerns of both local and regional users of the corridor. Providing your name and email is OPTIONAL. This information will only be available to the Project Team and will be used to add you to the project notification database. This information will NOT be available to other users.

Place a Comment on The Map

You can interact directly with the map by panning or zooming around. Once you have selected the desired location, simply click/tap to place your comment point. If you accidentally place a comment marker in the wrong location, you can "cancel" the placement and begin again. However, once comments are submitted, a user cannot go back to edit or change their response.

Map Based Navigation (On a Desktop PC)

There are several ways to interact with the map:

1. Use the basemap switcher to select the basemap you'd like to interact with
2. Hold down the shift key and drag a box on the map to zoom in
3. Hold Down Ctrl+Shift to zoom out
4. Roll your mouse roller to zoom in/out
5. Click the zoom in/out button

Map Based Navigation (On a Touch Screen Device)

There are two ways to interact with the map:

1. Pinch to zoom in/out
2. Tap the zoom in/out button

Adding Your Comment to the Map

Click directly on the map to pin your comment to a specific location. Then, provide details about the request and click 'Submit' to initiate your comment.

MyNLSD application is supported on the following browsers: Microsoft IE 9/10, Mozilla Firefox, Google Chrome and Apple Safari. The recommended browser is Google Chrome.
